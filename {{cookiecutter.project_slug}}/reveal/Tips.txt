# Tips for presentation

## Before

- Prepare the presentation
    - Introduction / Links 
    - Sommaire
    - Presentateur
    - Organisation
    - ...
    - Conclusion
    - Contacts
- Send presentation
- Virtual Room / question room / chatroom?
- Meeting request


## During

- Music before starting
- Connect to the virtual room
- Link to presentation / Wifi / Question room
- Presentation
- Presenter screen (notes / timer / pause mode)
- Water
- Mic ? (via phone?)


## After

- Thanks
- Resend presentation
